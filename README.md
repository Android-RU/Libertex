## Что такое Libertex для Android

Приложение Libertex для Android — это мобильная платформа для удобного доступа к рынкам и управления сделками прямо со смартфона. Либертекс позволяет быстро подключить брокерский счет, отслеживать цены и торговать различными активами в режиме реального времени.

* поддержка инструментов: валюта, акции, криптовалюта, сырьевые товары, ценные бумаги;
* возможность работать с лотами, видеть котировки в доллар и анализировать движение рынка;
* функции для трейдинг-операций: открытие сделок, управление рисками, контроль позиций.

Приложение подходит для пользователей, которые хотят совмещать мобильность и доступ к финансовым рынкам, включая опционы, облигации и фьючерсы.

* встроенные аналитические данные для принятия решений;
* удобный интерфейс для быстрого перехода между различными инструментами.

## Как скачать и установить libertex.apk на Андроид

Вы можете скачать libertex.apk из папки `release/` в репозитории и установить вручную:

```bash
adb install path/to/libertex.apk
```

> Убедитесь, что включена отладка по USB и разрешена установка из неизвестных источников.

| Параметр                       | Значение                                                       |
| ------------------------------ | -------------------------------------------------------------- |
| Минимальная версия Android     | Рекомендуемая актуальная версия Android (обычно от 7.0 и выше) |
| Объём памяти для установки     | Около 50–100 МБ свободного места                               |
| Размер установочного файла     | Зависит от версии в Google Play                                |
| Требуемое интернет-подключение | Постоянное, стабильное (Wi-Fi или мобильная сеть)              |
| Поддерживаемые устройства      | Большинство современных смартфонов и планшетов на Android      |

### Способ 1 — установка прямо на устройстве

1. **Перенесите файл на устройство**
Скопируйте `libertex.apk` в память телефона (например, в папку *Download*).

2. **Разрешите установку APK**

* Android 8.0 и выше:

 * Откройте **Настройки → Приложения и уведомления → Специальный доступ → Установка неизвестных приложений**.
 * Выберите приложение, через которое будете запускать APK (например, «Файлы»), и включите **«Разрешить из этого источника»**.
* Android 7.1 и ниже:

 * **Настройки → Безопасность → Неизвестные источники** → включите переключатель.

3. **Запустите установку**

* Откройте «Файлы» или любой файловый менеджер.
* Найдите `libertex.apk` и нажмите по нему.
* В появившемся окне выберите **Установить**.

4. **Запустите приложение**
После завершения установки нажмите **Открыть** или найдите ярлык на рабочем столе.

### Способ 2 — установка через ADB

1. **Включите режим разработчика и отладку по USB**

* **Настройки → О телефоне → Номер сборки** → нажмите 7 раз.
* В разделе «Для разработчиков» активируйте **Отладка по USB**.

2. **Установите ADB на компьютер**
Скачайте и распакуйте Android Platform Tools.

3. **Подключите устройство по USB**
Разрешите отладку по запросу на экране телефона.

4. **Выполните установку в терминале**

```bash
adb install -r libertex.apk
```

Ключ `-r` позволяет обновить уже установленную версию.

## Python скрипт для анализа Libertex.APK

Скрипт **libertex.py** позволяет получить базовую информацию об APK-файле приложения Libertex. Скрипт извлекает ключевые сведения, такие как имя пакета, версия, список разрешений, а также пытается сохранить иконку приложения.

**Возможности:**

* Чтение APK и извлечение метаданных
* Определение имени пакета, версии и названия приложения
* Получение списка разрешений
* Извлечение иконки приложения в формате PNG
* Поддержка вывода в текстовом и JSON-форматах

**Использование:**

```bash
python libertex.py --apk libertex.apk
```

**Опционально:**

```bash
python libertex.py --apk libertex.apk --json --verbose
```

**Требования:**

* Python 3.8+
* Зависимости:

```bash
pip install apkutils-patch Pillow
```
